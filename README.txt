Cara Connect Ke Web:
1. Open Command Prompt
2. Open folder C:\xampp\htdocs\fuzzy_flask
3. Copas Link folder fuzzy_flask : C:\xampp\htdocs\fuzzy_flask
4. Masuk Kembali ke command prompt
5. masukkan link folder ke-dalam command prompt: cd C:\xampp\htdocs\fuzzy_flask
6. masukkan : py app.py
7. setelah berhasil muncul ip nya maka lanjut akses di browser
8. masukkan link di browser : http://127.0.0.1:5000/
9. jika berhasil akan menampilkan tabel fuzzy_flask yang di coding dlm vscode
10. Selesai silahkan di masukkan input besar karbulator maka nanti akan menampilkan diagram kecepatan
